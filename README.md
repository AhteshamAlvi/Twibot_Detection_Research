# Twibot_Detection_Research

This repository studies how well bot-detection models transfer across Twitter/X datasets from different time periods.

## Logistic Regression Baseline

The full logistic regression workflow lives in:

- `notebooks/training/logistic_regression_training.ipynb`

That notebook contains the model implementation, feature preparation, experiment runs, result summaries, and paper-output generation in one place, matching the notebook-only style used for the Random Forest work.
